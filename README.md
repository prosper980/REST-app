# Youtube dashboard clone 

## REST Architecture Web Application


Setup: 

clone the repo

```bash
git git@github.com:prosper980/REST-app.git
```


Set environment variables
```bash
cd server
touch .env
cp .env-example .env
```

Set up the sever
```bash
docker compose up
```

Set up the client
```bash
cd client
docker compose up
```
