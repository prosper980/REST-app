# Youtube dashboard clone 

## REST Architecture Web Application


Setup: 

clone the repo

```bash
git clone https://github.com/prosper980/_dash.git 
```


Set environment variables
```bash
cd server
touch .env
cp .env-example .env
```

Set up the sever
```bash
docker compose up
```

Set up the client
```bash
cd client
docker compose up
```
